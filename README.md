# ①課題3-ラーメンを各要素で評価できるアプリ


## ②課題内容（どんな作品か）
ラーメンを週1回食べているので、いろんな店を相対的に評価したくなった。
Google Mapだと総合評価しか出ないので、各要素で評価ができるようにする
動きの説明
1：店とコメント記入
2：各種評価実施
3：ローカルストレージに格納
4：ストレージの数字を拾ってグラフ表示
